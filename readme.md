# Stats Preview Card Component

A responsive stats preview card built as part of the Frontend Mentor challenge.  
This project focuses on improving layout structuring, responsive design, and visual hierarchy using HTML and CSS.

---

## 📸 Screenshot

![Stats Preview Card](./screenshot.png)

---

## 🛠️ Built With

- HTML5
- CSS3
- Flexbox
- CSS Variables
- Responsive Design
- Mobile-first workflow

---

## 📌 Features

- Clean two-column layout (text + image section)
- Responsive design for mobile and desktop screens
- Styled statistics section with emphasis on key numbers
- Image blending/overlay effect for modern UI look
- Pixel-perfect implementation of challenge design
